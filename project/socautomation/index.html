SOC Automation Project Part 1

Overview

This is part of the SOC Automation Project. This part demonstrates how to setup a SIEM, SOAR, generate telemetry and automate response to security alerts. It also showcase how to deploy a virtual machine in Azure and setup it's security to ensure data privacy and integrity.

Project Goal

- Implement a SOC environment with Wazuh as the SIEM/XDR, TheHive for case management and integrate it to Shuffle for SOAR capabilities.

Tools and Technologies

- Wazuh
- Shuffle
- Windows 10 Pro
- Mimikatz

Wazuh and Sysmon Installation in Windows 10 Host

Objectives

-Install Wazuh in Azure VM instance and Sysmon in Windows 10 host.

What I did

1. Created a Virtual Machine instance in Azure for Wazuh.
2. Downloaded and installed Sysmon in the VirtualBox Windows 10 Host.

Skills Demonstrated

- Cloud Management
- Cloud Security
- Virtualization
- OS Management

Key Takeaway

This part of the project solidified my understanding on how to deploy VMs in the cloud and setup it's security,configuring secure access and firewall.
