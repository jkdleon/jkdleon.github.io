SOC Automation Project

Overview

This project demonstrates how to setup a SIEM, SOAR, generate telemetry and automate response in alerts.

Tools and Technologies

- Wazuh
- Shuffle
- TheHive
- Windows 10 Pro - Mimikatz

Project Goal

- Implement a SOC environment with Wazuh as the SIEM and XDR integrated to Shuffle for SOAR capabilities and TheHive for case management.

Wazuh and Sysmon Installation in Windows 10 Host

Objectives

-Install Wazuh in Azure VM instance and Sysmon in Windows 10 host.

What I did

1. Created a Virtual Machine instance in Azure for Wazuh.
2. Downloaded and installed Sysmon in the VirtualBox Windows 10 Host.

Skills Demonstrated

- Cloud Management
- Cloud Security
- Virtualization
- OS Management

Key Takeaway

This part of the project solidified my understanding on how to deploy VMs in the cloud and setup it's security,configuring secure access and firewall.
